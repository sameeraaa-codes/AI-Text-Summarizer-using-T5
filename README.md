# 🤖 AI Text Summarizer using T5 Transformer

An AI-powered **Text Summarization** web application built using **FastAPI**, **Hugging Face Transformers**, **PyTorch**, and the **T5 Transformer** model. The application generates concise and meaningful summaries from lengthy text using **abstractive text summarization**.

---

# 📌 Project Overview

This project allows users to summarize long paragraphs or articles through a simple and interactive web interface. The backend is developed using **FastAPI**, while the frontend is built using **HTML, CSS, and JavaScript**. The application leverages the **T5 Transformer model** to generate high-quality summaries in real time.

---

# ✨ Features

* 🤖 AI-powered abstractive text summarization
* ⚡ FastAPI REST API backend
* 🎨 Responsive and user-friendly interface
* 📝 Automatic text preprocessing
* 🚀 Real-time summary generation
* 💻 Supports CPU/GPU inference using PyTorch

# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/sameeraaa-codes/AI-Text-Summarizer-using-T5.git
```

## 2. Navigate to the Project Folder

```bash
cd AI-Text-Summarizer-using-T5
```

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

## 4. Run the Application

```bash
python -m uvicorn app:app --reload
```

# 🌐 Usage

Open your browser and visit:

```
http://127.0.0.1:8000/
```

API Documentation:

```
http://127.0.0.1:8000/docs
```

### Steps

1. Enter a paragraph or article into the text box.
2. Click the **Summarize** button.
3. Wait a few seconds while the model processes the text.
4. View the generated summary instantly.
